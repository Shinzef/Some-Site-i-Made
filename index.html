Introduction

C++ is a general-purpose programming language created by Danish computer scientist Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".

Throughout the years, C++ is updated 4 times in its lifetime with being the latest is C++ 20. With its 5th major update being C++ 23.

Some C++ features:

  • Object Oriented

    C++ is an object oriented language. It is a Object Oriented Programming language.

  • High-Level Language

Hello World !

First of all lets do the classic "Hello World!"

#include <iostream>

int main() {
    std::cout << "Hello World!" << std::endl;

    return 0;
}

Output:

Hello World!

So first lets explain this code.

#include <iostream> stands for "include iostream". iostream handles all the input and output.

int main() is the main function. It is the entry point of the program.

std::cout means character output. << is the operator for output. and std::endl is the end of line.

return 0 means return the number 0 to the operating system. This is the end of the program. And 0 mostly means success.

So what if we want it to print something else other than "Hello World"?

We can do that by using the "<<" operator. Then we can write what we want to print.

For example:

std::cout << "C++ Basics ?" << std::endl;

Data types

C++ has many data types.

  • Integers

    Integers are non fractional numbers.

  • Floats / Double

    Floats / Double are fractional numbers.

  • Characters

    Characters are single characters.

  • Strings

    Strings are sequences of characters.

  • Booleans

    Booleans are either true or false.

Each of these data types can be modified by using Type Modifiers.

  • signed

    Signed integers are positive and negative numbers.

  • unsigned

    Unsigned integers are positive numbers.

  • short

    Short integers are 16 bits(2 bytes). Used for small numbers(-32678 to 32678).

  • long

    Long integers are 32 bits(4 bytes). Used for large numbers(-2147483648 to 2147483648).

  • long long

    Long long integers are 64 bits(8 bytes). Used for large numbers(-9223372036854775808 to 9223372036854775808).

Operators

C++ has many operators.

  • Arithmetic Operators

    Arithmetic operators are used to perform arithmetic operations.

  • Assignment Operators

    Assignment operators are used to assign values to variables.

  • Relational Operators

    Relational operators are used to compare values.

  • Logical Operators

    Logical operators are used to combine conditions.

  • Bitwise Operators

    Bitwise operators are used to perform bitwise operations.

  • Type Conversion Operators

    Type conversion operators are used to convert values to different data types.

More about operators

This guide will only cover basic operators.

  • Addition

    Adds x and y

    x + y

  • Subtraction

    Subtracts y from x

    x - y

  • Multiplication

    Multiplies x and y

    x * y

  • Division

    Divides x by y

    x / y

  • Modulus

    Returns the remainder of x divided by y

    x % y

  • Increment

    Increments x by 1

    x++

  • Decrement

    Decrements x by 1

    x--

Some Examples: int x = 5;
int y = 3;
int z = x + y;
int a = x - y;
int b = x * y;
int c = x / y;
int d = x % y;
int e = x++;
int f = x--; Results:

z = 8
a = 2
b = 15
c = 1
d = 2
e = 6
f = 5

Type conversion

C++ has two types of type conversion.

  • Implicit

    Done by the compiler on its own, without any external trigger from the user.

  • Explicit

    Done by the user.
